조상 클래스: 부모, 상위(super), 기반(base) 클래스
자손 클래스: 자식, 하위 (sub), 파생된 (derived) 클래스

- 클래스간의 관계(맺는다면.. 종류는 다음과 같다)
1) 상속관계 : (is a 관계) 
2) 포함관계 : (has a 관계)
    예)
        원은 점이다: Circle extends Point 
        원은 점을 가지고 있다 : class Circle {point p = new Point()};

원은 도형이다 => ok 상속관계
