Map 인터페이스 (따로)
-------------------------------------------------------------

List : (중복허용) 순서가 있는 데이터의 집합
        구현 클래스 - ArrayList, LinkedList, Stack, Vector, ...
        1) ArrayList == 배열

Set : (중복허용안함) 순서를 유지하지 않는 데이터 집합
        구현 클래스 - HashSet, TreeSet, ...

Map : 키와 값의 쌍으로 이루어진 데이터 집합
        순서 유지 안함
        키 중복 불가, 값은 중복 허용
        구현 클래스 - HashMap, TreeMap, ...
